#  Albaik Karachi - High-Fidelity E-Commerce Experience

A premium, interactive food ordering platform built with **React.js**, focusing on micro-interactions, complex state management, and brand-driven UI/UX.

[View Live Demo]: ( https://al-baik-self-ordering-kiosk.vercel.app/ )

* **Dynamic Customization Engine:** Real-time spicy/regular toggles with "Heat Glow" UI feedback.
* **Smart Pricing Algorithm:** Automated logic for add-ons (e.g., first garlic sauce free, incremental pricing for extras).
* **Global State Management:** Utilizes React Context API to manage a persistent "Tray" (cart) system across all routes.
* **Simulated Order Tracking:** Real-time countdown timer using `useEffect` and `setInterval` hooks to simulate kitchen prep.
* **Performance UI:** Smooth CSS-in-JS animations, Glassmorphism navigation, and responsive "Hero Split" design.
